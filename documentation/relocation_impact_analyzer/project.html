relocation_impact_analyzer.project

Project class for the Relocation Impact Analysis Tool

The Project class is responsible for managing all aspects of a project within the relocation impact analysis tool. It encapsulates functionalities related to project setup, data management, and configuration handling. This includes creating new projects, loading and saving project configurations, managing data files (CSV format) for various aspects of the project (e.g., employee and office addresses, GPS coordinates), and ensuring the data integrity and structure conforms to expected formats. The class also provides utilities for accessing and manipulating project-specific data, supporting the broader analysis tasks by maintaining a central repository of the project's data and state.

217        """
218        Saves the current project's configuration and data files to disk.
219
220        This method serializes the project's attributes to a 'project.json' configuration file and saves any loaded

233        # save the attributes of the project class to the project file JSON
234        self.project = {}
235        for attr in self.project_file_attributes:
236            self.project[attr] = getattr(self, attr)
237        with open(self.project_file, "w") as f:
238            json.dump(self.project, f, indent=4)
239        # save all project data files that exist in the data dictionary
240        for file in self.data:
241            if self.data[file] is not None:
